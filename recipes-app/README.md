# Журка — Рецепты (Telegram Mini App)

Отдельное приложение: подбор рецептов под КБЖУ + корзина ингредиентов ВкусВилл.
Не зависит от приложения готовых блюд (pitanie-web).

## Что делает
1. Калькулятор КБЖУ (Харрис-Бенедикт × активность − 15%).
2. Недельное меню из рецептов гайдов (завтрак/обед/ужин с фото, масштаб под КБЖУ).
3. Готовка обедов/ужинов 2 раза в неделю (варочные сессии).
4. Корзина сырых ингредиентов ВкусВилл (копируемая ссылка).

## Запуск
    cd recipes-app
    pip install -r requirements.txt
    uvicorn main:app --host 0.0.0.0 --port 8001

Открыть http://СЕРВЕР:8001/ . В BotFather завести Mini App на этот URL.

## Файлы
- main.py — FastAPI (/api/kbju, /api/plan, /api/cart)
- kbju.py — расчёт КБЖУ
- planner.py — подбор недельного меню из рецептов
- vkusvill.py — поиск сырых ингредиентов + корзина (MCP)
- data/recipes.json — 74 рецепта из гайдов, data/img — 49 фото
- static/index.html — Mini App
